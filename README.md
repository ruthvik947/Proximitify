# Proximitify

- Web application that uses the Spotify, Google Maps and Eventful web APIs to identify and display concerts by artists the user may be interested in.
- Client side routing and front end rendering is implemented using Angular.js; communication with APIs (including web
authorization) and server side routing is implemented using Node.js and Express.js.
- In progress: currently working on deployment to Heroku and filtering/using the Eventful data being received.
